2-Player Dice Game

Competitive randomization logic for two players.

import random
p1 = random.randint(1, 6)
p2 = random.randint(1, 6)
print("Player 1 Rolled", p1)
print("Player 2 Rolled", p2)

if p1 > p2:
    print("Player 1 Wins")
elif p2 > p1:
    print("Player 2 Wins")
else:
    print("Its a tie")

2-Player Guessing Game

One player hides a number, the other attempts to find it.

p1 = int(input("Player 1: Hide your number: "))
print("\n" * 50) # Clears the screen
p2 = int(input("Player 2: Guess the number: "))

if p2 == p1:
    print("Right Guess! Player 2 wins")
else:
    print("Wrong Guess! Player 1 wins")

SimpleBot AI

An interactive chatbot loop for basic communication.

print("Chatbot: Hello! I am your AI Chatbot.")
while True:
    user = input("You: ").lower()
    if user == "hello":
        print("Chatbot: Hi! Nice to meet you")
    elif user == "your name":
        print("Chatbot: My name is SimpleBot.")
    elif user == "bye":
        break
